
To run the project go to steps:
 1- git clone https://github.com/enghesham/Todo_task.git 
 2- composer update
  3- copy .env.example and paste it in the file root and rename it .env 
  4- php artisan key:generate 
  5- php artisan migrate 
  6- yarn install 
  7- yarn run dev 
  8- php artisan serve
